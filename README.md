# 공공행사 VIP 실시간 상황관리 플랫폼

화성시 빅데이터팀 · AI·데이터 기반 행정혁신

## 빠른 시작 (로컬 개발)

```bash
npm install
npm run dev
```

## Vercel 배포 안내

이 저장소는 Vercel에 그대로 연결하면 자동 배포됩니다.

1. [vercel.com](https://vercel.com) 접속 → GitHub 로그인
2. [Add New Project] → 본 저장소 선택 → [Deploy]
3. 약 1~2분 후 배포 완료
4. 발급된 URL을 카카오톡으로 공유 가능

## 기술 스택

- React 18 + Vite 5
- Tailwind CSS 3
- Lucide React (아이콘)
